## 🌟 ABOUT BLACK-WIDOW

**Black-Widow** is a powerful, multi-functional WhatsApp bot built with the **Baileys MD API**. It's designed to enhance your WhatsApp experience with automated responses, media downloading, AI features, games, and much more. Whether you want to automate your group management, create custom stickers, or have fun with interactive features, Black-Widow has got you covered!

### 🎯 Key Highlights
- 🔒 **Multi-Device Support** - Works with latest WhatsApp MD version
- 🤖 **AI Integration** - Multiple AI models for image generation and chat
- 📥 **Advanced Downloader** - Download from 10+ platforms
- 🎮 **Gaming System** - Economy, levels, and interactive games
- 🎨 **Custom Stickers** - Create stickers from images, videos, or GIFs
- ⚡ **Lightning Fast** - Optimized for performance

## 🚀 QUICK SETUP

### 1️⃣ Get Your Credentials

You have two options to connect your WhatsApp:

#### Option A: Using Session ID (Recommended)
1. Visit [Session Generator](https://black-widow-pair.onrender.com)
2. Enter your WhatsApp number
3. Get your `SESSION_ID`
4. Save it for deployment

#### Option B: Using Pairing Code
1. Set `BOT_NUMBER` in `.env` or `config.js`
2. Deploy the bot
3. Copy the pairing code from terminal
4. Open WhatsApp > Linked Devices > Pair with code
